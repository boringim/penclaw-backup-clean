## 🔧 技术细节与问题解决

### 问题 1: 数据库 SQLite 锁死
- **现象**: backup-db.js 持续报 `SQLITE_BUSY`
- **根本原因**: Lancedb 数据库持续打开，SQLite `.backup()` API 无法获取排他锁
- **解决方案**: 改为 `fs.copyFileSync` 文件复制，绕过锁限制
- **结果**: ✅ 11/11 数据库备份成功

### 问题 2: ClawSec Windows 兼容性
- **现象**: `_pid_running()` 使用 `os.kill(pid, 0)` 在 Windows 上失败
- **解决方案**: 
  - 修改为 Windows 用 `tasklist /FI "PID eq {pid}"` 检查进程
  - 修改 PID 文件路径从 `/tmp/clawsec` 改为 `$env:TEMP\clawsec`
  - 修改 `cmd_stop` 用 `taskkill` 替代 `os.kill`
- **结果**: ✅ ClawSec 成功在 Windows 启动

### 问题 3: OpenClaw 工具阻塞
- **现象**: 使用 `exec(background: true)` 启动长期服务导致后续工具调用卡住
- **根本原因**: 后台会话永久占用，阻塞新调用
- **解决方案**: 不通过 OpenClaw 启动长期服务，改用独立 PowerShell 窗口手动启动
- **经验**: OpenClaw 不适合管理永久服务，应该用 Windows Service 或独立进程
