## Decisions
- **Whitelist over blacklist**: Only essential config files tracked in Git to avoid polluting history
- **SSH over HTTPS**: Credential-less automation, no credential manager needed
- **Continue-on-DB-failure**: File backups proceed even if DB snapshot fails
- **Hardcoded paths**: Eliminated .env dependency for Tier 2 to prevent misrouting
- **Exclude session locks**: Added `*.lock` to prevent snapshot failures from in-use files
